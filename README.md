# Basic Social Media App

Using -> HTML, CSS, JS, React 

## Installation process
1. #### Clone the repo using this command
    ```bash
    git clone https://github.com/hm-sezer/socialmedia.git
    ```
2. #### Install NPM Packages
    ```bash
    cd socialmedia
    npm install
    ```
3. #### And finally
    ```bash
    npm start
    ```
